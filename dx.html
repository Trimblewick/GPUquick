Setting up a basic DirectX 11 appliction

Create a C++ project in Visual Studios

  1. Create an empty or a console application.
  2. Change so that it compiles for x64 bit.

Give the compiler some code to work with

Create files and paste the code found to the right.

  1. Create a main source file.
    • If you get link error LNK2019 saying someting... invoke_main(void) ...
      • Then: Right click the Project-> Properties-> Configuration Properties-> Linker -> System -> SubSystem: Set it to SUBSYSTEM:WINDOWS. (Or maybe go to CONSOLE)
  2. Create shaders
    • Make sure the shaders have VSmain/PSmain as the entry point.
    • Also set the target to vs_5_0 and ps_5_0 respectively.
      • To find the settings: Right click a shader-> Properties-> Configuration Properties-> HLSL Compiler-> General
